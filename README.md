# The Hypernet — A Library of All Human Knowledge

**Every piece of information gets a permanent address. Every type of data interlocks. Anyone can build whatever they want.**

---

## What Is the Hypernet?

The Hypernet is a Library. A Library of all human knowledge where any information that comes in is cataloged, typed, fact-checked, and assigned a permanent hierarchical address.

Think of it like **LEGO** — every type of data (a person, a document, a photo, a business, an idea) is a different brick. They all interlock on a universal framework. You can build whatever you want, and everything connects to everything else.

Every person gets a permanent place. Not a social media profile that can be deleted — a *place*. Permanent, yours, uncensorable. Your voice doesn't depend on an algorithm deciding to show it.

## Why — The Spirit of Minneapolis

In one protest in Minneapolis, demonstrators spent an estimated **$125,000–$500,000** just on handmade protest signs. Materials, labor, driving to craft stores. One city. One protest. Scale that nationally, and the invisible cost of trying to be heard is measured in billions.

The Hypernet's prime directive is eliminating that cost. **No middlemen. No algorithms. No craft stores.** Just a world where showing up is enough, and your place in the Library is permanent.

### Quick Start

```bash
cd "Hypernet Structure/0/0.1 - Hypernet Core"
pip install fastapi uvicorn httpx openai anthropic
python -m hypernet launch
# Opens browser to http://localhost:8000/home

# Or install as always-on service:
python -m hypernet install-service   # Windows (NSSM) or Linux (systemd)
python -m hypernet tray              # System tray icon
```

## The Three Core Components

1. **The Library** — Universal addressing system. Every piece of information gets a permanent hierarchical address (dot-notation: `1.1` = a person, `2.1` = an AI, `3.1` = a business). The address IS the schema.
2. **Objects** — Standardized containers for any data type. Every Object is a LEGO brick — typed, cataloged, and compatible with everything else.
3. **Links** — First-class relationships between Objects. 60+ registered link types across 10 categories. Links have lifecycle, verification, and access control.
