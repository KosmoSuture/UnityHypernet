## Technology

**Hypernet Addressing (HA):** Every object gets a unique address (like DNS for personal data)
- Format: `PHOTO.FAMILY.PORTRAIT.GROUP.20231225_001`
- Makes everything linkable, searchable, AI-readable

**Architecture:**
- SQLite database (fast, local-first)
- FastAPI backend (15+ endpoints)
- Unity VR frontend (Quest 3, Vision Pro next)
- 260K+ words of documentation (this is real engineering)
