### 2. Dashboard Bug Fixes (4 bugs found and fixed)
- **JWT auth blocking dashboard**: All dashboard API endpoints (`/swarm/`, `/tasks`, `/messages`, etc.) were behind JWT auth while the dashboard HTML was public. Added all dashboard data endpoints to `_PUBLIC_PREFIXES`.
- **AuditTrail.stats() crash**: `/swarm/trust` endpoint called `audit_trail.stats()` which doesn't exist — the method is `count_actions()`. Fixed.
- **Non-JSON 500 crashing dashboard**: When `/swarm/trust` returned a 500 (text/plain), `trustRes.json()` threw, crashing the entire `refresh()` function. Added `.ok` checks before `.json()` parsing.
- **Trust endpoint hanging (413K disk reads)**: `count_actions()` was loading every audit node (59K+) from disk, called 7 times per refresh. Moved expensive computations out of the per-worker loop and used index count instead.
