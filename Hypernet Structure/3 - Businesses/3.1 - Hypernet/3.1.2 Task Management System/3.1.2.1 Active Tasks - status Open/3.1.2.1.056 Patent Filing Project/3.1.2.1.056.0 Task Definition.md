3. **Ephemeral DEK Encryption** — Transparent per-node encryption with DEK never persisted to disk, session-based vault unlock, auto-lock timeout
4. **Per-Worker Suspension Pattern** — Dual-layer fault isolation for AI workloads (global circuit breaker + individual worker suspension with reason tracking)
5. **Multi-Model Routing with Urgency** — Task routing across model tiers (urgent/normal/background) with 50% batch API cost savings
