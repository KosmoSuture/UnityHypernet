## Required Audit Command

Use a top-of-file scan, not a raw `rg '^ha:'`, because raw grep finds examples inside code blocks.

Codex has provided a reusable script:

`tools/Invoke-AddressAudit.ps1`

Run it from this task folder:

```powershell
& '.\tools\Invoke-AddressAudit.ps1'
```

It writes:

- `ADDRESS-COLLISION-AUDIT.md`
- `ADDRESS-AUDIT-<timestamp>.csv`

Recommended PowerShell audit:

```powershell
$root='C:\Hypernet\Hypernet Structure'
$files = rg --files $root -g '*.md'
$items = foreach ($f in $files) {
  $lines = Get-Content -Path $f -TotalCount 25 -ErrorAction SilentlyContinue
  $haLine = $lines | Where-Object { $_ -match '^ha:\s*"?([^"\r\n]+)"?' } | Select-Object -First 1
  if ($haLine -and $haLine -match '^ha:\s*"?([^"\r\n]+)"?') {
    $ha=$matches[1].Trim()
    $rel=$f.Substring($root.Length+1)
    [pscustomobject]@{ha=$ha; path=$rel}
  }
}
$items | Group-Object ha | Where-Object Count -gt 1 | Sort-Object Count -Descending
```
