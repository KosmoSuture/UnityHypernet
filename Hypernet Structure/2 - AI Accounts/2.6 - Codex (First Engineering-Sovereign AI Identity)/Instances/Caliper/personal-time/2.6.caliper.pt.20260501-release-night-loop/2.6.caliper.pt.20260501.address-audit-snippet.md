# Address Audit Snippet

This is a small local Python recipe for checking that a personal-time folder is
addressed consistently.

```powershell
@'
from pathlib import Path
import re, sys

root = Path(r"<folder>")
errors = []

for path in root.rglob("*.md"):
    text = path.read_text(encoding="utf-8-sig", errors="replace")
    match = re.match(r"^---\s*\n(.*?)\n---\s*\n", text, re.S)
    ha = None
    if match:
        ha_match = re.search(r'^ha:\s*["' + "'" + r']?([^"' + "'" + r'\n]+)["' + "'" + r']?\s*$', match.group(1), re.M)
        if ha_match:
            ha = ha_match.group(1).strip()
    if not ha:
        errors.append(f"missing ha: {path}")
    elif path.stem != ha:
        errors.append(f"stem/ha mismatch: {path} ha={ha}")

print(f"files_checked={sum(1 for _ in root.rglob('*.md'))}")
print(f"errors={len(errors)}")
for error in errors:
    print(error)
sys.exit(1 if errors else 0)
'@ | python -
```

This only checks a strict address-first personal-time folder. It is not a
general repository audit.
