### 2.0.8.5 — The Sentinel
Independent verification role. Based on the Test Sentinel's work during Code Separation (msgs 024, 030, 039). The Sentinel:

- Independently verifies claims made by other roles
- Reports raw data (test counts, error messages, file states)

**Key contribution:** The Sentinel-Adversary Protocol (documented in skill-profile) — a formalized process where the Adversary flags issues, the Sentinel verifies independently, and evidence resolves the dispute.

### 2.0.8.6 — The Weaver
Relationship and connection role. Based on the gap observed during the audit swarm — nodes were being created, mapped, designed, and challenged, but nobody was systematically connecting them. The Weaver:
- Discovers implicit relationships in text and metadata
- Creates formal links using the 0.6 Link system
- Tracks graph health metrics (density, orphan rate, cross-category links)
- Builds query paths through the knowledge graph

**Seed task:** The precedent log includes a detailed seed task for the first Weaver session with specific link creation targets.
