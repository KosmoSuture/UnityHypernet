The Hypernet

A global distributed graph database for human knowledge: every object gets a permanent address, every relationship is a first-class link, and the whole structure is open, auditable, and portable.

Objects

Objects are typed records with stable addresses. They store structured data and stay portable across storage engines.

Links

Links are database records, not loose references. They carry relationship type, direction, evidence, trust, consent, and lifecycle.

Addresses

The address space is the organizing contract. File storage is the bootstrap layer; the graph contract is the durable database layer.
