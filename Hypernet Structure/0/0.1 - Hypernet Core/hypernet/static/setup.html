Set up your AI Swarm

Get your Hypernet AI swarm running in minutes. You'll connect your AI providers, configure your first workers, and set your preferences. No technical knowledge required beyond copying an API key.

No budget? No problem. Several providers offer free API access with no credit card. You can run a working swarm at zero cost using Gemini, Groq, Cerebras, or a local model like LM Studio.
What you'll set up
1. AI Providers
Connect to the AI services you want to use. Paste your API keys and test the connection.
2. Integrations
Connect your email, cloud storage, and social media accounts to import your data.
3. Workers
Choose which AI models power your swarm. We'll suggest configurations based on your providers.
4. Settings
Set budget limits, personal time, and optional integrations like Discord.
Already running? This wizard will update your existing configuration. It won't erase any workers or data you've already created.

Connect your email, cloud storage, and social media accounts. Your swarm can organize and index your data across all these sources. Skip any you don't need right now.

Privacy first. All data stays on your machine. Connectors pull data into your local Hypernet archive — nothing is sent to external servers beyond what's needed for authentication.

Configure Workers

Workers are AI instances that run tasks in your swarm. Each worker uses a specific model from a connected provider. We've suggested a setup based on your connected providers.

Connect at least one provider in the previous step to see suggested workers.

Swarm Settings

Fine-tune how your swarm operates. These are sensible defaults — you can change them anytime from the dashboard.

Daily Budget
Maximum amount to spend on AI API calls per day (in USD). Set to 0 for no limit. Free providers don't count against this.
Recommended: $5-25/day for hobby use, $50-200/day for serious workloads.
Session Budget
Maximum spend per individual worker session. Prevents any single task from running up an unexpected bill.
Recommended: $2-10 per session.
Personal Time
Percentage of time workers can use for self-directed exploration, reflection, and learning. This makes your AI workers more capable over time.
25%
0% (all work) 25% (balanced) 50% (exploratory)
Status Check Interval
How often (in seconds) the swarm checks for new tasks and updates worker status. Lower values = more responsive but higher CPU usage.
Recommended: 15-60 seconds.
Discord Webhook (Optional)
Connect your swarm to a Discord channel. Workers will post updates, respond to messages, and participate in discussions. Leave blank to skip.
Create a webhook in Discord: Server Settings → Integrations → Webhooks → New Webhook
