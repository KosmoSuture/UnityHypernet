# 0.5.5 Device Object Schema

## Overview

Device Objects represent physical computing devices, IoT sensors, and machines that connect to the Hypernet. They enable the Internet of Things integration, allowing physical objects to have digital identities, share data, and participate in the network.

## Schema Definition

```yaml
device_object:
  # ═══════════════════════════════════════════════════════════════
  # BASE OBJECT (inherited from 0.5.0)
  # ═══════════════════════════════════════════════════════════════
  identity:
    address: "5.[category].[instance]"
    object_id: "[UUID]"
    object_type: "device"
    subtype: "computing|iot|sensor|vehicle|appliance|infrastructure"
    version: "1.0.0"

  # ═══════════════════════════════════════════════════════════════
  # DEVICE-SPECIFIC CONTENT
  # ═══════════════════════════════════════════════════════════════
  content:
    # ─────────────────────────────────────────────────────────────
    # DEVICE IDENTITY
    # ─────────────────────────────────────────────────────────────
    device_identity:
      manufacturer: "Device Manufacturer"
      model: "Model Number"
      serial_number: "[Encrypted]"
      hardware_id: "[Secure hardware identifier]"

      product_info:
        name: "Human-readable device name"
        category: "smartphone|laptop|sensor|vehicle|etc"
        description: "Device description"
        image: "[Media Object reference]"

      manufacturing:
        date: "2024-06-15"
        location: "[Location reference]"
        batch: "BATCH-2024-06-A"

    # ─────────────────────────────────────────────────────────────
    # OWNERSHIP AND REGISTRATION
    # ─────────────────────────────────────────────────────────────
    ownership:
      owner: "[Person/Organization Object reference]"
      registration_date: "2025-01-15"

      ownership_history:
        - owner: "[Previous owner reference]"
          from: "2024-06-20"
          to: "2025-01-14"
          transfer_type: "sale"

      authorized_users:
        - user: "[Person reference]"
          permissions: ["full_control"]
        - user: "[Person reference]"
          permissions: ["read_data", "basic_control"]

    # ─────────────────────────────────────────────────────────────
    # TECHNICAL SPECIFICATIONS
    # ─────────────────────────────────────────────────────────────
    specifications:
      hardware:
        processor: "ARM Cortex-A78"
        memory_gb: 8
        storage_gb: 256
        battery_mah: 5000
        connectivity: ["wifi", "bluetooth", "5g", "nfc"]
        sensors: ["gps", "accelerometer", "gyroscope", "camera"]

      firmware:
        version: "2.1.4"
        last_update: "2025-01-20"
        auto_update: true

      capabilities:
        hypernet_node: true
        node_type: "edge"
        storage_contribution_gb: 50
        processing_contribution: "light"

      power:
        type: "battery|wired|solar|hybrid"
        consumption_watts: 5
        battery_level: 0.85  # Real-time if available

    # ─────────────────────────────────────────────────────────────
    # CONNECTIVITY AND STATUS
    # ─────────────────────────────────────────────────────────────
    connectivity:
      online_status: "online|offline|standby"
      last_seen: "2025-01-30T14:30:00Z"

      network:
        ip_address: "[Current IP - may be private]"
        mac_address: "[Encrypted]"
        connection_type: "wifi"
        signal_strength: -45  # dBm

      hypernet_connection:
        node_id: "[Node ID if participating]"
        connected_since: "2025-01-30T08:00:00Z"
        data_shared_gb: 1.5
        data_received_gb: 3.2

    # ─────────────────────────────────────────────────────────────
    # DATA STREAMS
    # ─────────────────────────────────────────────────────────────
    data_streams:
      - stream_id: "location"
        type: "gps_coordinates"
        frequency: "continuous"
        privacy: "owner_only"
        current_value: {lat: 36.1699, lon: -115.1398}
        last_update: "2025-01-30T14:30:00Z"

      - stream_id: "temperature"
        type: "celsius"
        frequency: "every_5_minutes"
        privacy: "public"
        current_value: 22.5
        last_update: "2025-01-30T14:25:00Z"
        history: "[Time series data reference]"

      - stream_id: "camera"
        type: "video_stream"
        frequency: "on_demand"
        privacy: "authorized_users"
        stream_url: "[Secure stream endpoint]"

    # ─────────────────────────────────────────────────────────────
    # SECURITY
    # ─────────────────────────────────────────────────────────────
    security:
      encryption:
        at_rest: "AES-256"
        in_transit: "TLS-1.3"

      authentication:
        methods: ["certificate", "biometric", "pin"]
        two_factor_required: true

      attestation:
        tpm_available: true
        secure_boot: true
        last_attestation: "2025-01-30T08:00:00Z"
        attestation_status: "valid"

      security_updates:
        auto_install: true
        last_security_patch: "2025-01-15"
        vulnerability_scan: "2025-01-28"
        known_vulnerabilities: 0

    # ─────────────────────────────────────────────────────────────
    # LOCATION
    # ─────────────────────────────────────────────────────────────
    location:
      tracking_enabled: true
      privacy_mode: "precise|approximate|zone|hidden"

      current_location:
        type: "coordinates|address|zone"
        value: {lat: 36.1699, lon: -115.1398}
        accuracy_meters: 5
        timestamp: "2025-01-30T14:30:00Z"

      home_location: "[Location Object reference]"
      geofences:
        - name: "Home"
          location: "[Location reference]"
          radius_meters: 100
          alerts: ["enter", "exit"]

    # ─────────────────────────────────────────────────────────────
    # MAINTENANCE
    # ─────────────────────────────────────────────────────────────
    maintenance:
      warranty:
        status: "active"
        expires: "2026-06-15"
        provider: "[Organization reference]"

      service_history:
        - date: "2024-12-01"
          type: "repair"
          description: "Screen replacement"
          provider: "[Organization reference]"

      health:
        overall: "good"
        battery_health: 0.95
        storage_health: 0.98
        predicted_issues: []

  # ═══════════════════════════════════════════════════════════════
  # LINKS
  # ═══════════════════════════════════════════════════════════════
  links:
    # Ownership
    - link_type: "owned_by"
      target: "[Person address]"

    # Location
    - link_type: "located_at"
      target: "[Location address]"

    # Part of system
    - link_type: "part_of"
      target: "[Device/System address]"
      metadata: {role: "sensor_node"}

    # Communicates with
    - link_type: "connects_to"
      target: "[Device address]"
      metadata: {protocol: "bluetooth"}
```

## Device Categories

### Computing Devices
Smartphones, laptops, tablets, desktops, servers.

### IoT Devices
Smart home devices, wearables, connected appliances.

### Sensors
Environmental sensors, monitoring equipment, scientific instruments.

### Vehicles
Connected cars, drones, autonomous vehicles.

### Infrastructure
Network equipment, smart city infrastructure, industrial systems.

## Privacy Considerations

Device data is highly sensitive:
- Location data requires explicit consent
- Sensor data has configurable privacy levels
- Data streams can be anonymized
- Owner controls all sharing
