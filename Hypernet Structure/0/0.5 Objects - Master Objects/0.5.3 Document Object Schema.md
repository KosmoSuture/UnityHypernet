# 0.5.3 Document Object Schema

## Overview

Document Objects contain textual and structured information—from simple notes to complex research papers, from configuration files to legal contracts. They are the primary carriers of human knowledge in the Hypernet.

## Schema Definition

```yaml
document_object:
  # ═══════════════════════════════════════════════════════════════
  # BASE OBJECT (inherited from 0.5.0)
  # ═══════════════════════════════════════════════════════════════
  identity:
    address: "4.[category].[subcategory].[instance]"
    object_id: "[UUID]"
    object_type: "document"
    subtype: "text|structured|code|legal|academic|technical"
    version: "1.0.0"

  # ═══════════════════════════════════════════════════════════════
  # DOCUMENT-SPECIFIC CONTENT
  # ═══════════════════════════════════════════════════════════════
  content:
    # ─────────────────────────────────────────────────────────────
    # DOCUMENT METADATA
    # ─────────────────────────────────────────────────────────────
    document_metadata:
      title: "Document Title"
      subtitle: "Optional Subtitle"
      abstract: "Brief summary of the document"

      authorship:
        authors:
          - person: "[Person Object reference]"
            role: "primary_author"
            contribution: "Wrote sections 1-3"
          - person: "[Person Object reference]"
            role: "contributor"
            contribution: "Reviewed and edited"

        ai_assisted: true
        ai_disclosure: "AI used for initial draft generation"

      classification:
        category: "technical_documentation"
        tags: ["hypernet", "architecture", "specification"]
        keywords: ["distributed systems", "privacy", "encryption"]
        language: "en"
        reading_level: "advanced"

      dates:
        drafted: "2025-01-15"
        published: "2025-01-30"
        last_reviewed: "2025-01-30"
        review_due: "2025-07-30"

      status:
        publication_status: "draft|review|published|archived"
        review_status: "unreviewed|peer_reviewed|expert_reviewed"
        accuracy_confidence: 0.95

    # ─────────────────────────────────────────────────────────────
    # DOCUMENT BODY
    # ─────────────────────────────────────────────────────────────
    body:
      format: "markdown|html|latex|plain|structured"

      # For text documents
      text_content:
        raw: "[Full document text]"
        rendered: "[Rendered HTML/display format]"

      # For structured documents
      structured_content:
        schema: "[Schema reference]"
        data: {}

      # Sections (for navigable documents)
      sections:
        - section_id: "1"
          title: "Introduction"
          anchor: "introduction"
          content: "[Section content]"
          subsections: []

        - section_id: "2"
          title: "Background"
          anchor: "background"
          content: "[Section content]"
          subsections:
            - section_id: "2.1"
              title: "Historical Context"
              content: "[Subsection content]"

    # ─────────────────────────────────────────────────────────────
    # EVIDENCE AND CITATIONS
    # ─────────────────────────────────────────────────────────────
    evidence:
      citations:
        - citation_id: "1"
          type: "reference"
          target: "[Document Object reference]"
          context: "Supporting claim in section 2"
          quote: "Relevant quote from source"

        - citation_id: "2"
          type: "data_source"
          target: "[External URL or Object]"
          accessed: "2025-01-25"
          archived: "[Archive reference]"

      claims:
        - claim_id: "c1"
          statement: "Specific claim made in document"
          section: "3.2"
          evidence: ["citation_id: 1", "citation_id: 2"]
          verification_status: "verified|unverified|disputed"
          verifiers: ["[Person references who verified]"]

      trust_score:
        overall: 0.87
        methodology: "Based on citation quality and verification"
        breakdown:
          citation_quality: 0.9
          peer_review: 0.85
          author_reputation: 0.88
          age_factor: 0.95

    # ─────────────────────────────────────────────────────────────
    # VERSIONING
    # ─────────────────────────────────────────────────────────────
    versioning:
      current_version: "2.1.0"
      version_history:
        - version: "1.0.0"
          date: "2025-01-15"
          author: "[Person reference]"
          changes: "Initial publication"

        - version: "2.0.0"
          date: "2025-01-25"
          author: "[Person reference]"
          changes: "Major revision of section 3"
          diff: "[Diff Object reference]"

      branching:
        branches:
          - name: "experimental"
            base_version: "2.0.0"
            status: "active"
        merge_requests: []

    # ─────────────────────────────────────────────────────────────
    # ACCESS AND LICENSING
    # ─────────────────────────────────────────────────────────────
    licensing:
      license: "CC-BY-4.0"
      license_url: "https://creativecommons.org/licenses/by/4.0/"
      commercial_use: true
      derivative_works: true
      attribution_required: true

      restrictions:
        - type: "geographic"
          description: "Not available in certain jurisdictions"
        - type: "temporal"
          embargo_until: "2025-06-01"

  # ═══════════════════════════════════════════════════════════════
  # LINKS
  # ═══════════════════════════════════════════════════════════════
  links:
    # Authorship
    - link_type: "authored_by"
      target: "[Person address]"
      metadata: {role: "primary"}

    # Citations
    - link_type: "cites"
      target: "[Document address]"
      metadata: {context: "Supporting evidence"}

    # Related documents
    - link_type: "related_to"
      target: "[Document address]"
      metadata: {relationship: "complementary"}

    # Supersedes
    - link_type: "supersedes"
      target: "[Document address]"
      metadata: {reason: "Updated information"}

    # Part of collection
    - link_type: "part_of"
      target: "[Collection address]"
      metadata: {sequence: 3}
```

## Document Subtypes

### Text Documents
- Articles, essays, reports
- Markdown or rich text format
- Full-text searchable

### Structured Documents
- JSON, YAML, XML data
- Schema-validated
- Queryable fields

### Code Documents
- Source code files
- Syntax highlighting
- Executable (in sandboxed environments)

### Legal Documents
- Contracts, agreements
- Version-controlled
- Signature tracking

### Academic Documents
- Research papers
- Peer review tracking
- Citation networks

### Technical Documents
- Specifications
- API documentation
- Architecture diagrams

## Truth and Evidence

### Citation Requirements
Documents making factual claims should:
- Link to primary sources
- Provide verifiable evidence
- Disclose methodology

### Trust Scoring
Trust scores are calculated based on:
- Citation quality and quantity
- Peer review status
- Author reputation
- Community verification
