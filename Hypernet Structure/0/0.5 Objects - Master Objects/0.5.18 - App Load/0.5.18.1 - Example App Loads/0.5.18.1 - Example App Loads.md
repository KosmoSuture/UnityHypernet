# 0.5.18.1 - Example App Loads

This folder contains draft and reviewed examples of `0.5.18` App
Load objects. These are concrete manifests for Hypernet-aware
applications.

Current examples:

- `0.5.18.1.1 - Personal Assistant App Load.md` - draft app-load
  instance for Matt's Android/Wear OS personal assistant app.

Example app-loads are not Official unless their manifest says
`integrity.verified: true` and includes an Official attestation
chain.
