# 0.5.4 Media Object Schema

## Overview

Media Objects contain audiovisual content—images, audio, video, and 3D models. They are essential for the VR-native experience of the Hypernet and carry rich metadata for accessibility, attribution, and discovery.

## Schema Definition

```yaml
media_object:
  # ═══════════════════════════════════════════════════════════════
  # BASE OBJECT (inherited from 0.5.0)
  # ═══════════════════════════════════════════════════════════════
  identity:
    address: "6.[category].[instance]"
    object_id: "[UUID]"
    object_type: "media"
    subtype: "image|audio|video|3d_model|interactive"
    version: "1.0.0"

  # ═══════════════════════════════════════════════════════════════
  # MEDIA-SPECIFIC CONTENT
  # ═══════════════════════════════════════════════════════════════
  content:
    # ─────────────────────────────────────────────────────────────
    # MEDIA METADATA
    # ─────────────────────────────────────────────────────────────
    media_metadata:
      title: "Media Title"
      description: "Description of the media content"

      creation:
        creator: "[Person/Organization Object reference]"
        creation_date: "2025-01-30"
        creation_tool: "Tool or device used"
        ai_generated: false
        ai_disclosure: "If AI-assisted, describe how"

      classification:
        category: "photography|illustration|music|speech|film|animation|model"
        genre: "documentary"
        tags: ["nature", "landscape", "mountain"]
        content_rating: "G|PG|PG-13|R|NC-17"
        sensitive_content:
          violence: false
          nudity: false
          language: false
          other: []

    # ─────────────────────────────────────────────────────────────
    # TECHNICAL SPECIFICATIONS
    # ─────────────────────────────────────────────────────────────
    technical:
      # Common to all media
      file_format: "png|jpg|mp3|mp4|glb|etc"
      file_size_bytes: 15728640
      checksum:
        algorithm: "sha256"
        value: "[Hash value]"

      # Image-specific
      image:
        width_px: 3840
        height_px: 2160
        aspect_ratio: "16:9"
        color_space: "sRGB"
        bit_depth: 8
        has_alpha: false
        dpi: 300

      # Audio-specific
      audio:
        duration_seconds: 245.5
        sample_rate_hz: 48000
        bit_depth: 24
        channels: 2
        codec: "FLAC"
        bitrate_kbps: 1411

      # Video-specific
      video:
        duration_seconds: 3600
        width_px: 3840
        height_px: 2160
        frame_rate: 60
        video_codec: "H.265"
        audio_codec: "AAC"
        bitrate_kbps: 25000
        has_audio: true
        has_subtitles: true
        hdr: true

      # 3D Model-specific
      model_3d:
        format: "glTF|USDZ|FBX|OBJ"
        polygon_count: 50000
        texture_count: 5
        rigged: true
        animated: true
        animations: ["walk", "run", "idle"]
        lod_levels: 4
        vr_optimized: true

    # ─────────────────────────────────────────────────────────────
    # RENDITIONS (multiple quality levels)
    # ─────────────────────────────────────────────────────────────
    renditions:
      - rendition_id: "original"
        purpose: "archival"
        storage_reference: "[Chunk reference]"

      - rendition_id: "high"
        purpose: "high_quality_display"
        resolution: "3840x2160"
        storage_reference: "[Chunk reference]"

      - rendition_id: "medium"
        purpose: "standard_display"
        resolution: "1920x1080"
        storage_reference: "[Chunk reference]"

      - rendition_id: "low"
        purpose: "preview|thumbnail"
        resolution: "480x270"
        storage_reference: "[Chunk reference]"

      - rendition_id: "thumbnail"
        purpose: "listing"
        resolution: "150x84"
        storage_reference: "[Chunk reference]"

    # ─────────────────────────────────────────────────────────────
    # ACCESSIBILITY
    # ─────────────────────────────────────────────────────────────
    accessibility:
      alt_text: "Descriptive text for images"

      # For audio/video
      transcripts:
        - language: "en"
          format: "srt"
          content_reference: "[Document Object reference]"
        - language: "es"
          format: "srt"
          content_reference: "[Document Object reference]"

      captions:
        - language: "en"
          type: "closed"
          content_reference: "[Caption track reference]"

      audio_description:
        available: true
        track_reference: "[Audio track reference]"

      sign_language:
        available: true
        language: "ASL"
        track_reference: "[Video track reference]"

    # ─────────────────────────────────────────────────────────────
    # RIGHTS AND LICENSING
    # ─────────────────────────────────────────────────────────────
    rights:
      copyright_holder: "[Person/Organization reference]"
      copyright_year: 2025
      license: "CC-BY-NC-4.0"

      usage_rights:
        personal_use: true
        commercial_use: false
        modification: true
        distribution: true
        attribution_required: true

      model_releases:
        - subject: "[Person reference]"
          release_document: "[Document reference]"
          scope: "full"

      location_releases:
        - location: "[Location reference]"
          release_document: "[Document reference]"

    # ─────────────────────────────────────────────────────────────
    # VR/SPATIAL PROPERTIES
    # ─────────────────────────────────────────────────────────────
    spatial:
      # For images that can be placed in VR
      display_mode: "flat|curved|spherical|volumetric"
      suggested_size:
        width_meters: 2.0
        height_meters: 1.125

      # For 3D models
      scale: 1.0
      default_position: {x: 0, y: 0, z: 0}
      default_rotation: {x: 0, y: 0, z: 0}
      bounding_box:
        min: {x: -1, y: 0, z: -1}
        max: {x: 1, y: 2, z: 1}

      # Interaction
      interactive: true
      interaction_points:
        - name: "info_point"
          position: {x: 0, y: 1.5, z: 0}
          action: "show_info"
          target: "[Document reference]"

  # ═══════════════════════════════════════════════════════════════
  # LINKS
  # ═══════════════════════════════════════════════════════════════
  links:
    # Creator
    - link_type: "created_by"
      target: "[Person address]"

    # Subject matter
    - link_type: "depicts"
      target: "[Person/Location/Object address]"
      metadata: {role: "subject"}

    # Part of collection
    - link_type: "part_of"
      target: "[Collection address]"

    # Derived from
    - link_type: "derived_from"
      target: "[Media address]"
      metadata: {transformation: "cropped and color-corrected"}
```

## Media Types

### Images
Static visual content: photographs, illustrations, diagrams, screenshots.

### Audio
Sound content: music, speech, podcasts, sound effects, ambient audio.

### Video
Moving visual content: films, recordings, animations, screen captures.

### 3D Models
Three-dimensional objects: avatars, environments, products, sculptures.

### Interactive
Media with user interaction: VR experiences, interactive diagrams, simulations.

## VR Integration

All media is designed to be viewable in VR:
- Images can be placed on virtual walls or as floating panels
- Videos play on virtual screens
- 3D models exist as objects in virtual space
- Audio has spatial positioning
