# 0.4.4.4.0 - About Review

A Review is an evaluation of content quality, accuracy, or compliance by one or more reviewers. Unlike an Approval (which is a binary gate), a Review produces feedback, scores, and revision requests.

## Review vs Approval

- **Approval**: Yes/No decision gate. Blocks progress until decided.
- **Review**: Qualitative evaluation. Produces feedback. May iterate.

Reviews are used for peer review of documents, code review, content verification, and trust score calculation.
