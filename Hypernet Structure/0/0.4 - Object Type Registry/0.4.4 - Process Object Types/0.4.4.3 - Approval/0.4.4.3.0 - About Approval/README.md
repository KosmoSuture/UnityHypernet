# 0.4.4.3.0 - About Approval

An Approval is a decision gate within a workflow that requires one or more authorized entities to approve or reject before the process continues. Approvals enforce the Hypernet's governance principles.

## Multi-Instance Review

Per the AI Data Protection Standard (2.0.19), destructive operations require 3-instance review from 2+ accounts. Approval objects track who approved, when, and under what authority.
