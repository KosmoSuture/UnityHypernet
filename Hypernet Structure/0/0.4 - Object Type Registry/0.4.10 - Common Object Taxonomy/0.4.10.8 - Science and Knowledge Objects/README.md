# 0.4.10.8 - Science and Knowledge Objects

Concepts, claims, evidence, methods, and models.

## Object Types

| Address | Type | Purpose |
|---|---|---|
| `0.4.10.8.1` | Concept | An abstract idea, category, pattern, or meaning unit. |
| `0.4.10.8.2` | Claim | A statement that can be supported, contradicted, or revised. |
| `0.4.10.8.3` | Evidence | An observation, source, record, or artifact supporting a claim. |
| `0.4.10.8.4` | Question | An unresolved inquiry seeking one or more answers. |
| `0.4.10.8.5` | Answer | A response to a question with support, confidence, and scope. |
| `0.4.10.8.6` | Hypothesis | A testable proposed explanation or prediction. |
| `0.4.10.8.7` | Method | A repeatable research, measurement, or analysis procedure. |
| `0.4.10.8.8` | Measurement | A quantified observation with unit, method, and uncertainty. |
| `0.4.10.8.9` | Citation | A structured reference to a source. |
| `0.4.10.8.10` | Model | A conceptual, mathematical, AI, or computational representation. |

## Modeling Rule

Objects in this domain store typed data. Meaning comes from links: ownership, provenance, containment, permission, temporal validity, and semantic relation should be modeled as first-class links wherever possible.
