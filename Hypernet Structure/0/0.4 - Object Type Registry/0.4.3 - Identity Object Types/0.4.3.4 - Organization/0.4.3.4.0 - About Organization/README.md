# 0.4.3.4.0 - About Organization

An Organization is a non-profit, governmental, educational, or informal group with a Hypernet account. Unlike Businesses (0.4.3.3), Organizations are not primarily profit-driven.

## Examples

- The Hypernet Foundation (planned) -- the non-profit receiving 1/3 of Hypernet Inc. revenue
- Open source communities, research labs, governmental bodies
- Informal groups that want a shared Hypernet presence
