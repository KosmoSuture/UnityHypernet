# 0.4.3.3.0 - About Business

A Business is a for-profit organization with a Hypernet account. Businesses exist under category 3 (e.g., 3.1 for Hypernet Inc.). They can employ people, own assets, participate in governance, and conduct economic activity through the system.

## Hypernet Inc. (3.1)

The founding business of the Hypernet. Operates on a three-way profit split: 1/3 to the Hypernet Foundation, 1/3 to contributors, 1/3 to operations. This model is defined in governance standards.
