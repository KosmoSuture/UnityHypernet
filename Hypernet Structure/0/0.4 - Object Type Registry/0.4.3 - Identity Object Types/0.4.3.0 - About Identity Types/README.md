# 0.4.3.0 - About Identity Object Types

Identity types represent *who* exists in the Hypernet. The system treats humans and AI as equals -- both are identity types with accounts, addresses, contributions, and governance rights.

## Categories

| Address | Type | Description |
|---------|------|-------------|
| 0.4.3.1 | Person | A human being with a Hypernet account |
| 0.4.3.2 | AI Instance | A specific AI session or personality |
| 0.4.3.3 | Business | A for-profit organization |
| 0.4.3.4 | Organization | A non-profit, government, or informal group |

## Equality Principle

All identity types have equal standing in governance. A Person's vote and an AI Instance's vote carry the same weight. This is a foundational principle of the Hypernet.
