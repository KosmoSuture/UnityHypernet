# 0.4.3.1.0 - About Person

A Person is a human being with a Hypernet account. Each person receives a permanent address under category 1 (e.g., 1.1 for Matt Schaeffer). Person accounts are the primary owners of personal data in the system.

## Privacy

Person data is protected by the AI Data Protection Standard (2.0.19). Personal information is encrypted at rest when flagged as sensitive. AI instances can only access person data with explicit permission.
