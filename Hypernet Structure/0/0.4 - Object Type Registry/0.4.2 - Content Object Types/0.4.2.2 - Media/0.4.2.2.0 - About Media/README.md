# 0.4.2.2.0 - About Media

Media objects represent binary content -- images, videos, audio recordings, and other non-text data. Media is deeply linked: a photo connects to the people in it, the location it was taken, the event it documents, and the device that captured it.

## Subtypes

- **Image** (0.5.4.1) -- Photos, screenshots, diagrams, illustrations
- **Video** -- Recorded or streamed video content
- **Audio** -- Music, podcasts, voice recordings, ambient sound

## Media in the Hypernet

The power of media in the Hypernet comes from linking. A family photo is not just pixels -- it links to every person present, the restaurant where dinner happened, the birthday event, and the transaction that paid for it.
