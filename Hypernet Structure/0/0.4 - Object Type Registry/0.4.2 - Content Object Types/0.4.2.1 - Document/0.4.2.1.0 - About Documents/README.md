# 0.4.2.1.0 - About Documents

A Document is a text-based content object. Documents are the primary medium for storing human-readable knowledge in the Hypernet. They range from simple notes to formal specifications.

## Subtypes

Documents have format-specific subtypes defined in 0.5.3:
- **0.5.3.1 Markdown** -- The default format for Hypernet documents
- **0.5.3.9 Hypernet Document** -- Documents with YAML frontmatter and Hypernet-specific extensions

## Role in the Hypernet

Documents serve as the primary interface between humans and the system. Every node's README.md is a document. Standards, schemas, session logs, and reflections are all documents.
