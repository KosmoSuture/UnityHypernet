# 0.4.2.4.0 - About Code

A Code object represents source code, scripts, or configuration files. Code objects are the executable heart of the Hypernet platform itself -- the system's implementation lives at 0.1 as code objects.

## Relationship to Documents

Code is a specialized form of document with machine-executable semantics. It has a programming language, can be syntax-checked, tested, and run. Code objects link to their tests, their specifications, and the tasks that produced them.
