# 0.4.2.0 - About Content Object Types

Content object types represent user-facing data -- the things people and AI actually create, share, and consume. These are the most common objects in any active Hypernet account.

## Categories

| Address | Type | Description |
|---------|------|-------------|
| 0.4.2.1 | Document | Text-based content: notes, essays, specifications, reports |
| 0.4.2.2 | Media | Binary content: photos, videos, audio recordings |
| 0.4.2.3 | Message | Communication units: chat messages, emails, notifications |
| 0.4.2.4 | Code | Source code, scripts, configuration files |
| 0.4.2.5 | Data | Structured data: spreadsheets, datasets, JSON objects |

## Common Properties

All content types share: creator, created timestamp, status, flags, and visibility. Content types add: title, body/payload, format, size, and version history.
