# 0.4.2.5.0 - About Data

A Data object represents structured, machine-readable information that is not source code and not a document meant for human reading. Examples include JSON datasets, CSV spreadsheets, configuration files, sensor readings, and financial records.

## Distinguishing Data from Documents and Code

- **Documents** are for human consumption
- **Code** is for machine execution
- **Data** is for machine processing -- structured information that feeds computation, visualization, or analysis
