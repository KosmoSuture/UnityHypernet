# 0.4.2.3.0 - About Messages

A Message is a communication unit sent from one entity to another. Messages are the atoms of conversation -- they include chat messages, emails, Discord posts, SMS, and any other form of person-to-person or person-to-group communication.

## Distinguishing Messages from Documents

- A **Document** is standalone content meant to persist and be referenced
- A **Message** is contextual communication, part of a conversation thread

Messages link to their sender, recipient(s), conversation thread, and any referenced content.
