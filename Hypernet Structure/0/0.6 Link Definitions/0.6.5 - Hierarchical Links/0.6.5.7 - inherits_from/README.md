# 0.6.5.7 - inherits_from

**Category:** Hierarchical (0.6.5)
**Directed:** Yes | **Transitive:** Yes
**Inverse:** inherited_by

## Description

Type inheritance. The source type inherits properties from the target type. Used in the object type hierarchy. Example: Image (0.5.4.1) inherits_from Media (0.5.4) inherits_from Master Object (0.5.0).
