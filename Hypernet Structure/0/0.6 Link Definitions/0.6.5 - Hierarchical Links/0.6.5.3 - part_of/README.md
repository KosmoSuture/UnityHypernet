# 0.6.5.3 - part_of

**Category:** Hierarchical (0.6.5)
**Directed:** Yes | **Transitive:** Yes
**Inverse:** has_part

## Description

Compositional relationship. The source is a component or part of the target. Transitive -- if A is part of B and B is part of C, then A is part of C. Distinct from child_of which implies address hierarchy; part_of implies conceptual composition.
