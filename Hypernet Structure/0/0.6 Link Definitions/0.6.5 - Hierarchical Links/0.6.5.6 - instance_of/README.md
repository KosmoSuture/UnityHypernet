# 0.6.5.6 - instance_of

**Category:** Hierarchical (0.6.5)
**Directed:** Yes | **Transitive:** No | **Cardinality:** many-to-one
**Inverse:** has_instance

## Description

Type instantiation. The source is a concrete instance of the target type. Example: Matt's birthday photo is an instance_of the Image type (0.5.4.1).
