# 0.6.5.4 - contains (hierarchical)

**Category:** Hierarchical (0.6.5)
**Directed:** Yes | **Transitive:** Yes | **Cardinality:** one-to-many
**Inverse:** contained_in

## Description

Logical containment in a hierarchy. The source logically contains the target. Distinct from the spatial contains (0.6.4.2) which implies physical space.
