# 0.6.2.0 - About Organizational Links

Organizational links define relationships between people and organizations, and between organizations themselves. They structure employment, membership, corporate hierarchy, partnerships, and supply chains.

## Link Types

| Address | Link Type | Description |
|---------|-----------|-------------|
| 0.6.2.1 | member_of | General membership in an organization |
| 0.6.2.2 | employed_by | Employment relationship |
| 0.6.2.3 | founded | Founding relationship |
| 0.6.2.4 | contributes_to | Contribution to org or project |
| 0.6.2.5 | subsidiary_of | Corporate ownership hierarchy |
| 0.6.2.6 | partners_with | Strategic partnership |
| 0.6.2.7 | competes_with | Market competition |
| 0.6.2.8 | affiliated_with | General affiliation |
| 0.6.2.9 | supplies | Supply chain relationship |
| 0.6.2.10 | regulates | Regulatory relationship |
