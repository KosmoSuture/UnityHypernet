# 0.6.2.4 - contributes_to

**Category:** Organizational (0.6.2)
**Directed:** Yes | **Symmetric:** No | **Transitive:** No
**Source:** person | **Target:** organization, project
**Inverse:** has_contributor

## Description

Contribution relationship. Tracks ongoing contributions to an organization or project, especially important for the Hypernet's three-way profit split model.

## Properties

- **contribution_type**: code | content | design | management | support | financial
- **since**: Start date
- **total_hours / credit_hours**: For profit sharing calculations
- **active**: Whether currently contributing
