# 0.6.6.1 - similar_to

**Category:** Semantic (0.6.6)
**Directed:** No | **Symmetric:** Yes | **Transitive:** No

## Description

Semantic similarity. Two objects share significant characteristics or meaning. Not transitive -- A similar to B and B similar to C does not imply A similar to C (drift).
