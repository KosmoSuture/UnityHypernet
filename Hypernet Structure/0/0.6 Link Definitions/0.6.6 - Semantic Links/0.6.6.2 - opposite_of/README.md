# 0.6.6.2 - opposite_of

**Category:** Semantic (0.6.6)
**Directed:** No | **Symmetric:** Yes | **Transitive:** No

## Description

Semantic opposition. Two concepts are antonyms or represent opposing ideas. Example: "open" opposite_of "closed".
