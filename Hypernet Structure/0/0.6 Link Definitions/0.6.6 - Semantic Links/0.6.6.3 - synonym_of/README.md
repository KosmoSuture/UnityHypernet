# 0.6.6.3 - synonym_of

**Category:** Semantic (0.6.6)
**Directed:** No | **Symmetric:** Yes | **Transitive:** Yes

## Description

Semantic equivalence. Two concepts mean the same thing. Transitive -- if A is synonym of B and B synonym of C, then A is synonym of C.
