# 0.6.6 Semantic Links

## Overview

Semantic links encode meaning-based relationships between objects. Where hierarchical links describe structure and reference links describe citations, semantic links capture the conceptual connections that make knowledge navigable: similarity, opposition, analogy, implication, and context.

These are the links that enable the Hypernet's emergent intelligence — the ability to surface unexpected connections and build understanding across domains.

## Similarity Links

### similar_to
Objects that share characteristics.

```yaml
link_type:
  name: "similar_to"
  address: "0.6.6.similar_to"
  category: "similarity"
  directed: false
  symmetric: true
  transitive: false      # Similarity is NOT transitive (A~B, B~C does not imply A~C)

  source_types: ["*"]
  target_types: ["*"]
  cardinality: "many_to_many"

  properties:
    similarity_type: "structural|functional|visual|conceptual|behavioral"
    dimensions: ["list of dimensions they're similar on"]
    strength: "0.0-1.0"    # Degree of similarity
    method: "manual|embedding|clustering|ai_inference"

  verification:
    method: "community_consensus"
    can_be_inferred: true   # AI can suggest similarity links

  notes: |
    Similarity links are often inferred by AI through:
    - Embedding cosine similarity
    - Structural comparison
    - Behavioral pattern matching
    Inferred links should carry their confidence score and method.
```

## Semantic Link Properties

### Inference Rules

Semantic links enable knowledge inference:

| Rule | From | Infer |
|------|------|-------|
| **Synonym transitivity** | A synonym_of B, B synonym_of C | A synonym_of C |
| **Example classification** | X example_of Concept | X instance_of Concept (weak) |
| **Implication chain** | A implies B, B implies C | A implies C |
| **Contradiction propagation** | A contradicts B, B corroborates C | A weakly contradicts C |
| **Context inheritance** | A part_of B, C contextualizes B | C contextualizes A (weaker) |

### Confidence Propagation

When semantic links are inferred rather than manually created, confidence decays:

```
Direct observation:     1.0 * link_strength
One hop inference:      0.7 * link_strength
Two hop inference:      0.5 * link_strength
Three+ hop inference:   0.3 * link_strength (minimum threshold)
```

Links inferred below 0.3 confidence are not created automatically but may be suggested for human/AI review.
