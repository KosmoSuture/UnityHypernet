# 0.6.4.2 - contains (spatial)

**Category:** Spatial & Temporal (0.6.4)
**Directed:** Yes | **Symmetric:** No | **Transitive:** Yes
**Inverse:** contained_in

## Description

Spatial containment. The source physically or logically contains the target. Transitive -- if A contains B and B contains C, then A contains C.
