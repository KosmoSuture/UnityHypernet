# 0.6.4.0 - About Spatial and Temporal Links

Spatial and temporal links anchor objects in physical space and time. They enable location-based discovery, historical analysis, causal reasoning, and VR navigation.

## Spatial Links (0.6.4.1-0.6.4.5)

Location, containment, proximity, adjacency, and geographic origin.

## Temporal Links (0.6.4.6-0.6.4.11)

Sequence, simultaneity, causation, scheduling, and deadlines.

## VR Navigation Links (0.6.4.12-0.6.4.13)

Portals and line-of-sight for the WebXR spatial browser.
