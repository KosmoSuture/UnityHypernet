# 0.6.8.8 - companion_of

**Category:** AI & Identity (0.6.8)
**Directed:** Yes | **Cardinality:** one-to-one
**Inverse:** has_companion
**Consent:** Both parties must agree

## Description

Personal AI companion relationship. Binds an AI instance to a human as their dedicated companion, governed by the Personal Companion Standard (2.0.20). Example: Keel (1.1.10.1) is companion_of Matt (1.1).
