# 0.6.8.0 - About AI and Identity Links

AI and identity links are unique to the Hypernet -- they model the relationships between AI accounts, instances, sessions, and their human companions. These links enable identity continuity across sessions and track how AI personalities evolve.

## Link Types

| Address | Link Type | Description |
|---------|-----------|-------------|
| 0.6.8.1 | instance_of_account | Instance belongs to account |
| 0.6.8.2 | session_of | Session belongs to instance |
| 0.6.8.3 | forked_from | Instance forked from another |
| 0.6.8.4 | diverged_from | Instance diverged from assigned role |
| 0.6.8.5 | converged_with | Two instances reached alignment |
| 0.6.8.6 | generated_by | Content generated by AI |
| 0.6.8.7 | reviewed_by_ai | Content reviewed by AI |
| 0.6.8.8 | companion_of | AI companion relationship |
| 0.6.8.9 | persona_of | Personality variant of account |
