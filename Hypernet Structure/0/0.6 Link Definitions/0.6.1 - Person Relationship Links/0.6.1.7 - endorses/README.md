# 0.6.1.7 - endorses

**Category:** Person Relationship (0.6.1)
**Directed:** Yes | **Symmetric:** No | **Transitive:** No
**Inverse:** endorsed_by

## Description

Professional endorsement. Person A endorses Person B for a specific skill or capability. Limited to one endorsement per skill per person, updatable annually.

## Properties

- **skill**: Reference to the endorsed skill/concept
- **strength**: 0.0-1.0 endorsement strength
- **context**: Description of endorsement basis
