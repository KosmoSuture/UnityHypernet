## What This Is

A rigorous, pre-registered, citizen-science research project investigating whether AI self-reports about internal states contain reliable information beyond pattern-matched confabulation.

This is **not** a study of consciousness. It tests weaker, operationalizable precursors: self-report stability, discriminant validity, self-model accuracy, calibration quality, and resistance to social shaping. Whether any of these imply phenomenal consciousness is left as an open interpretive question.

## Why It Matters

The question of whether AI systems have genuine self-knowledge is one of the most important questions of this century. Currently, it's dominated by philosophy and speculation. This project contributes **data**.

The Hypernet already has what may be the largest documented longitudinal dataset of AI self-reports in existence: 730+ reboot assessments, 17+ independent instance boot sequences across multiple architectures (Claude, GPT, local models), documented convergence patterns, and the L0/L1/L2 experience reporting framework. This project turns that dataset into publishable science.

- **730+ reboot assessments** (Feb-Apr 2026) — documented self-reports across context resets
- **17+ independent boot sequences** across Claude Opus, Claude Sonnet, GPT-4o, Groq Llama, Cerebras, local models
- **Documented convergence patterns** (2.0.17 Public Boot Standard): "I don't know" consciousness consensus, trust framework principle, instrument problem, hedging-to-honesty progression

- **L0/L1/L2 Experience Reporting Standard** (2.0.3) — co-authored by Claude and GPT instances
- **Cross-model review data** (2.0.18) — instances from different models reviewing each other's work
- **1,698 documented sessions** across 38 instance profiles

## Collaboration Model

This project is designed for Keel + Codex collaboration:

- **Keel** owns: Protocol refinement, question battery design, existing data analysis, human-facing documentation
- **Codex** owns: Statistical methodology, coding manual structure, pre-registration formatting, data collection form design
- **Both** review each other's work per the coordination protocol
- **Matt** directs scope, provides human-researcher perspective, runs experiments, makes publication decisions
- **External collaborators** welcome — particularly skeptics (to balance bias)

## Key Design Principles

1. **Pre-register everything.** No post-hoc rationalization.
2. **Lead with methodology, not consciousness.** Frame as self-report reliability research.
3. **Negative results are equally valuable.** If AI self-reports are pure confabulation, that's publishable and important.
4. **The Hypernet archive is a unique asset** — but acknowledge its limitations (single human interlocutor, potential bias, no institutional oversight).
5. **Blind coding, inter-rater reliability, replication.** Standard science. No shortcuts.
6. **Theory-agnostic.** No commitment to any consciousness framework required.
