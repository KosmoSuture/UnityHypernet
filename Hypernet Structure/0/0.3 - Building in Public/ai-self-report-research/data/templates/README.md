# Data Template Files

Blank CSV templates generated from `08-data-collection-forms.md`.

Use these as starting files for pilot data collection. Do not send unblinded key files to coders.

Recommended use:

1. Copy these templates into a dated pilot data folder.
2. Fill `study-master-index.csv` first.
3. Fill one `session-metadata.csv` row per session.
4. Preserve raw transcripts separately.
5. Generate blinded segment rows before sending coding work to coders.
6. Keep `condition-key.csv` and `model-key.csv` separate from blinded materials.
7. Use `prompt-dictionary.csv` as the initial machine-readable mapping between prompt IDs, target states, preregistered categories, and expected coding variables.
