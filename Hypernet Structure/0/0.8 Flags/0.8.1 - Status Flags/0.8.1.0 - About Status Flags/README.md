# 0.8.1.0 - About Status Flags

Status flags indicate the trust level and accuracy state of content. These are the most impactful flags in the reputation system -- verifying content stakes your reputation on its accuracy.

## Flags

| Address | Flag | Visual | Description |
|---------|------|--------|-------------|
| 0.8.1.1 | verified | Green checkmark | Content verified as accurate |
| 0.8.1.2 | disputed | Yellow warning | Accuracy being challenged |
| 0.8.1.3 | false | Red X | Determined to be false after review |
| 0.8.1.4 | needs-review | Blue eye | Content should be reviewed |
| 0.8.1.5 | help-requested | Orange hand | Creator requesting help |
