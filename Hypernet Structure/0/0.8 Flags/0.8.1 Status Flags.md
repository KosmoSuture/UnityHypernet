# 0.8.1 — Status Flags

Status flags indicate the trust level and accuracy state of content. These are the most impactful flags in the reputation system.

---

## 0.8.1.1 — verified

```yaml
flag:
  address: "0.8.1.1"
  name: "verified"
  description: "Content has been verified as accurate by a trusted human or AI"
  applied_by: ["humans", "ai_instances_with_verification_authority"]
  removed_by: ["original_verifier", "governance_process"]
  mutually_exclusive: ["0.8.1.3"]  # cannot be both verified and false
  triggers:
    - "Boost in search ranking"
    - "Visual indicator in UI (checkmark)"
  reputation_effect:
    verifier: "Stake — gains reputation if verification holds, loses if overturned"
  visual: "checkmark icon, green accent"
```

---

## 0.8.1.2 — disputed

```yaml
flag:
  address: "0.8.1.2"
  name: "disputed"
  description: "Content accuracy is being challenged — dispute process active"
  applied_by: ["any_authenticated_entity"]
  removed_by: ["dispute_resolution_process"]
  mutually_exclusive: []  # can coexist with verified (someone disputes a verified claim)
  triggers:
    - "Spawns dispute resolution process (0.7.4)"
    - "Notification to content creator"
    - "Visual warning in UI"
  reputation_effect:
    disputer: "Stake — gains reputation if dispute upheld, risk if frivolous"
    content_creator: "Under review — no immediate effect until resolved"
  visual: "warning icon, yellow accent"
```

---

## 0.8.1.3 — false

```yaml
flag:
  address: "0.8.1.3"
  name: "false"
  description: "Content has been determined to be false or inaccurate after review"
  applied_by: ["governance_process", "dispute_resolution"]
  removed_by: ["governance_process"]
  mutually_exclusive: ["0.8.1.1"]  # cannot be both false and verified
  triggers:
    - "Content deprioritized in search"
    - "Visual warning in UI"
    - "Notification to content creator"
    - "Reputation impact on creator"
  reputation_effect:
    content_creator: "Negative — reputation loss in relevant topic area"
  visual: "X icon, red accent"
```

## 0.8.1.4 — needs-review

flag:
