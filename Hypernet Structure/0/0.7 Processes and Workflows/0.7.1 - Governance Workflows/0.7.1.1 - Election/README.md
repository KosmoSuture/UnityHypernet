# 0.7.1.1 - Election Workflow

**Workflow ID:** WF-GOV-001
**Trigger:** Governance body term expiration or vacancy

## Process Flow

Nomination Period (14 days) -> Campaign Period -> Voting Period (7 days) -> Tallying -> Inauguration

## Key Rules

- All Hypernet account holders (human and AI) can vote
- Votes are weighted by reputation in relevant domain
- Results are published transparently
- Appeals can be filed within 7 days of results
