# 0.7.1.3 - Policy Change Workflow

**Workflow ID:** WF-GOV-003
**Trigger:** Approved proposal requiring policy modification

## Process Flow

Proposal Approved -> Draft Change -> Review Period -> Implementation -> Verification

## Key Rules

- Policy changes must reference the approved proposal
- Review period allows affected parties to raise concerns
- Changes are versioned with full revision history
- Rollback procedure defined for each change
