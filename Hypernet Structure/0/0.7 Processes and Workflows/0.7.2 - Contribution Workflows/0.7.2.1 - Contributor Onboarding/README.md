# 0.7.2.1 - Contributor Onboarding

**Workflow ID:** WF-CONTRIB-001
**Trigger:** New contributor registration

## Process Flow

Registration -> Identity Verification -> Access Provisioning -> Orientation -> First Assignment

## Key Rules

- Contributors get a permanent Hypernet address
- Access level starts at T0 (read-only) and escalates with trust
- Orientation covers governance, contribution guidelines, and standards
