# 0.7.2.3 - Credit Calculation

**Workflow ID:** WF-CONTRIB-003
**Trigger:** Revenue distribution period (monthly)

## Process Flow

Aggregate Work Logs -> Weight by Type and Quality -> Calculate Shares -> Publish Breakdown -> Appeal Period

## Key Rules

- Code, documentation, review, and community engagement all earn credit
- Quality multiplier based on peer review scores
- AI contributions are weighted equally to human contributions
- Transparent breakdown published for all contributors
