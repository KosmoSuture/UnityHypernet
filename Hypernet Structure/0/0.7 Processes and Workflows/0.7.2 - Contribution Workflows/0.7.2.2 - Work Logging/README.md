# 0.7.2.2 - Work Logging

**Workflow ID:** WF-CONTRIB-002
**Trigger:** Contributor completes work

## Process Flow

Log Work -> Link to Outputs -> Peer Verification -> Credit Applied

## Key Rules

- All work must be linked to its outputs (code, docs, reviews)
- AI work is automatically logged via session records
- Human work requires self-reporting with peer verification
- Disputed claims go through the review workflow (0.7.3)
