# 0.7.2.0 - About Contribution Workflows

Contribution workflows manage how people and AI participate in building the Hypernet. They track work, calculate credit, and distribute the three-way profit split.

## Workflows

| Address | Workflow | Description |
|---------|----------|-------------|
| 0.7.2.1 | Contributor Onboarding | Welcoming and integrating new contributors |
| 0.7.2.2 | Work Logging | Recording and verifying work performed |
| 0.7.2.3 | Credit Calculation | Computing each contributor's share |
| 0.7.2.4 | Payment Distribution | Distributing revenue to contributors |
