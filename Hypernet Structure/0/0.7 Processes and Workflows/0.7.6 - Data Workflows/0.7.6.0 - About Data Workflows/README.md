# 0.7.6.0 - About Data Workflows

Data workflows handle the lifecycle of data -- from import through active use to archival. They ensure data quality, provenance tracking, and proper retention.

## Workflows

| Address | Workflow | Description |
|---------|----------|-------------|
| 0.7.6.1 | Import Pipeline | Bringing external data into the Hypernet |
| 0.7.6.2 | Export Pipeline | Extracting data for external use |
| 0.7.6.3 | Archive and Retention | Long-term storage and lifecycle management |
| 0.7.6.4 | Migration | Moving data between stores or formats |
