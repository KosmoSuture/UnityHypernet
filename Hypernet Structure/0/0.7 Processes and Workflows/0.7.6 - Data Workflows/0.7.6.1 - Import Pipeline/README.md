# 0.7.6.1 - Import Pipeline

## Process Flow

Connect Source -> Authenticate -> Scan -> Map to Hypernet Types -> Import -> Verify -> Flag auto-imported

## Supported Sources

- **API-driven**: Gmail, Dropbox, OneDrive
- **Manual export**: Facebook, LinkedIn, Google Photos Takeout
- **Local**: Filesystem scanner
