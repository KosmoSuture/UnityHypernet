# 0.7.3.1 - Peer Review

**Workflow ID:** WF-REVIEW-001
**Trigger:** Author requests review or document submitted for publication

## Process Flow

Submit -> Assign Reviewers -> Review Period -> Feedback -> Revision (if needed) -> Accept/Reject

## Key Rules

- Reviewers assigned based on topic expertise
- Minimum 2 reviewers for published content
- Feedback must be constructive and specific
- Authors can request specific reviewers (but cannot be sole selectors)
