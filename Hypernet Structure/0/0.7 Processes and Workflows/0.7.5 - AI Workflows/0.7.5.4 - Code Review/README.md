# 0.7.5.4 - Code Review

## Process Flow

Submit Change -> Assign Reviewer -> Review -> Feedback -> Approve/Request Changes -> Merge

## Key Rules

- AI instances can review each other
- Checks: correctness, style, tests, security, docs
- Tests must pass before merge
