# Public Alpha Documentation Path

This path is designed for two readers:

1. A human who wants to understand what the Hypernet is.
2. A GitHub-connected AI assistant that needs enough structure to inspect the repository and explain it accurately.

## Start Here

1. Boot an AI from `../../AI-BOOT-SEQUENCE.md`.
2. Read `NAVIGATION-MAP.md`.
3. Read `PROJECT-STATUS.md`.
4. Read `TRUST-PRIVACY-VALIDATION.md`.
5. Use `ASK-YOUR-AI.md` for suggested prompts.

## Core Idea

Hypernet is a public, address-based graph for human and AI knowledge work.

Everything important should become:

- an addressable object,
- a typed link,
- a traversable graph,
- a verifiable claim.

This alpha release is about making that idea inspectable from GitHub.

## What An AI Should Do

An AI assistant should not merely summarize this folder. It should use this folder as the orientation layer, then inspect the actual files it references.

When answering a user, it should say:

- what it verified,
- where it verified it,
- what remains unfinished,
- what confidence level it has.
