# Process-Loads — Specialized Knowledge Modules

This folder is the specialty-knowledge layer of the Public Alpha
Grand Tour. Each file here is a "process-load" — a focused
~2,000-3,000 token module that the Tour Guide loads on demand
when a user wants depth in a specific area, instead of loading
the entire Hypernet on boot.

## Process-Load Files

Each file carries its own `ha` under
`0.3.public-alpha.grand-tour.process-load.<slug>`:

| File | Address Slug | Topic |
|---|---|---|
| `0.3.public-alpha.grand-tour.process-load.architecture.md` | `architecture` | Graph database, addresses, objects, links, API |
| `0.3.public-alpha.grand-tour.process-load.privacy.md` | `privacy` | Lockers, mandalas, aliases, E2E encryption, AI sentries |
| `0.3.public-alpha.grand-tour.process-load.democracy.md` | `democracy` | Knowledge consensus, dispute lifecycle, reputation |
| `0.3.public-alpha.grand-tour.process-load.ai-governance.md` | `ai-governance` | 2.* origin, instances, standards, role framework |
| `0.3.public-alpha.grand-tour.process-load.business-onboarding.md` | `business-onboarding` | 3.* companies, migration, AI assistance |
| `0.3.public-alpha.grand-tour.process-load.geospatial-vr.md` | `geospatial-vr` | Mesh, VR layer, spatial addressing |
| `0.3.public-alpha.grand-tour.process-load.public-stewardship.md` | `public-stewardship` | Forks, attestation, governance-of-governance |
| `0.3.public-alpha.grand-tour.process-load.economics.md` | `economics` | Funding, ownership, profit-sharing, foundation |
| `0.3.public-alpha.grand-tour.process-load.personal-ai-swarm.md` | `personal-ai-swarm` | User-side AI helpers, security sentries |

## Format Convention

See `../0.3.public-alpha.grand-tour.process-load-standard.md` for the structural template
each process-load follows (overview, scope, current state,
extensions, related files).

## Why Address This Folder Explicitly

Per Matt's directive (2026-05-01): every folder under `0.3.docs/`
must be Hypernet-addressable. This folder previously inherited
its identity from its parent Grand Tour folder; the explicit
address `0.3.public-alpha.grand-tour.process-loads` makes the
folder a first-class node so it can be linked to and audited
directly.
