# Ask Your AI

Use these prompts with any GitHub-connected AI assistant that can read this repository.

## Boot Prompt

```text
Open this repository and read AI-BOOT-SEQUENCE.md. Boot as a Hypernet Guide.
Take me through the Grand Tour, then load the process-load that best matches
my questions. Explain from repository evidence and separate what is implemented,
documented, planned, and unknown.
```

## Single-Link Expertise Prompt

```text
I am giving you one GitHub repository link and one boot file: AI-BOOT-SEQUENCE.md.
Use that boot file to become a starter Hypernet expert. Do not rely on memory or
sales language. Load the Grand Tour, use the module menu, fetch the relevant
process-loads, inspect source files for claims, and cite exact paths. When you
answer, separate implemented, documented, planned, and unknown.
```

## Trust Prompt

```text
Verify the Hypernet privacy and trust claims. Read 0.3.docs/0.3.public-alpha/0.3.public-alpha.trust-privacy-validation.md,
then inspect the referenced code and docs. Tell me which claims are implemented,
which are documented only, and which are still planned.
```

## Skeptic Reframe Prompt

```text
The Hypernet's #1 stated goal is "be the #1 most trusted company in the world
for all of history." That sounds grandiose. Don't try to convince me. Instead,
help me reframe my evaluation:

(1) Is this *actually possible* — could a system built the way this one is
    being built (open code, hash-authenticated objects, audit logs as
    first-class records, fork-aware mode flags, AI-verifiable claims) reach
    the trust standard the project claims it could?

(2) If it IS possible — even partially — what good could it do for the
    world? List specific outcomes by sector (privacy, governance, AI safety,
    research, civic infrastructure).

(3) Given (1) and (2), what are the right questions to ask next? The honest
    project status is: messy, running off a laptop and a Dell desktop,
    Bitcoin-era seed stage, ugly but real. Treat that as data, not as a
    reason to dismiss.

Read README.md, 0.3.docs/0.3.public-alpha/0.3.public-alpha.md,
0.3.docs/0.3.public-alpha/0.3.public-alpha.project-status.md,
and the public-stewardship process-load. Then walk me through your
reframe.
```
