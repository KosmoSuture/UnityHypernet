## **TASK OBJECT SCHEMA (3.1.2.0)**

Each task is an Object with this structure:

Task Object:  
├── General Metadata  
│   ├── Hypernet Address (3.1.2.1.XXX)  
│   ├── Task ID (unique identifier)  
│   ├── Created Date/Time  
│   ├── Created By (Person Object link)  
│   ├── Status (Open/Assigned/In Progress/Blocked/Complete)  
│   ├── Priority (Critical/High/Medium/Low)  
│   └── Tags/Keywords  
├── Task-Specific Metadata  
│   ├── Title  
│   ├── Description  
│   ├── Estimated Hours  
│   ├── Actual Hours (tracking)  
│   ├── Skills Required (array)  
│   ├── Deliverables (array)  
│   ├── Acceptance Criteria  
│   ├── Due Date (if applicable)  
│   └── Budget (if applicable)  
├── Relationship Links  
│   ├── Assigned To (Person Objects)  
│   ├── Upstream Dependencies (Task Objects)  
│   ├── Downstream Dependents (Task Objects)  
│   ├── Related Documents (Document Objects)  
│   ├── Related Discussions (Discussion Objects)  
│   └── Parent Task (for subtasks)  
└── Tracking Data  
    ├── Activity Log  
    ├── Comments/Updates  
    ├── File Attachments

    └── Status Change History

## **SKILL-BASED TASK DISCOVERY SYSTEM**

For your 80 man-hours of team in Mexico/Nicaragua, here's how they find work:

### **Interface Concept:**

1. **Skill Filter View**: "I have these skills: \[X, Y, Z\]"  
   * System shows all tasks requiring those skills  
   * Sorted by priority and availability  
2. **Time Availability View**: "I have X hours available this week"  
   * System recommends tasks matching time estimate  
3. **Learning Path View**: "I want to learn \[skill\]"  
   * System shows tasks that teach that skill with mentorship  
4. **Dependency View**: "Show me what's ready to work on"  
   * Only shows tasks where upstream dependencies are complete

### **Communication Framework:**

* **Upstream Communication**: "I'm blocked because I need X from the team working on Y"  
* **Downstream Communication**: "I've completed this, notifying all dependent tasks"  
* **Lateral Communication**: "I need help with \[aspect\], looking for someone with \[skill\]"
